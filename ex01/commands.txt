===
Название: grep
Описание: Печатает строки, которые включают в себя паттерн(или удовлетворяют регулярному выражению)
Синтаксис: grep [опции] паттерн
Пример: ls | grep .txt
Примечание: -E для регулярных выражений
===
Название: eval
Описание: объединяет аргументы в одну строку, которую использует в качестве входных данных для оболочки и выполнения команд
Синтаксис: eval [аргументы]
Пример: sa="$(ssh-agent -s)" &&  eval $sa

===
Название: touch
Описание: создает пустой файл
Синтаксис: touch [опции] файл
Пример: touch ex.txt
Примечание: если файл уже существует, команда обновит его временные метки, не затрагивая содержимое файла
===
Название: mv
Описание: перемещает или переименовывает файлы
Синтаксис: mv [опции] откуда куда
Пример: mv my_photo.png screenshot.png
Примечание: можно использовать для переименования файлов
===
Название: diff
Описание: сравнивает содержимое файлов построчно
Синтаксис: diff [опции] файлы
Пример: diff ex.txt requirements.txt
===
Название: whereis
Описание: показывает полный путь к программе
Синтаксис: whereis [-bmsu] [-BMS directory... -f] filename...
Пример: whereis ros2
===
Название: hostname
Описание: показывает DNS-домен/IP-адрес устройства
Синтаксис: hostname [опции]
Пример: hostname -I
===
Название: zip
Описание: архивирует файлы
Синтаксис: zip [options] название_архива.zip название_файла
Пример: zip -r txt_file.zip ex.txt
===
Название: df
Описание: показывает сколько места свободно/занято на диске
Синтаксис: df [опции] [файл]
Пример: df -h .
Примечание: -h показывает в понятном для человека виде
===
Название: du 
Описание: показывает размер папок и файлов в текущей директории
Синтаксис: du [опции] [файл]
Пример: du -sh 
Примечание: -s показывает общий вес директории
